# node_koa_blog

### 目录结构

* config 配置文件
* dblib sql 操作文件
* middlewares 中间价
* public 静态资源文件
* routes 路由
* views html 模版
* index 入口文件
* package.json 

### 模块功能
1. `koa` 框架
2. `koa-bodyparser` 表单解析中间件
3. `koa-mysql-session`、`koa-session-minimal` 数据库中间件
4. `koa-router` 路由
5. `koa-static` 静态资源加载中间件
6. `ejs` 模版引擎
7. `moment ` date中间件
8. `mysql` 数据库
9. `koa-views` 模版呈现中间件
